Privacy & keys

Local data, clear controls

Your household data stays local to your device. CookAIware stores everything in simple local JSON files.

Local storage

CookAIware stores inventory, meal plans, and family profile data locally. No household data is uploaded to external services.

No assistant memory

The LLM does not store state. It reads and writes only to the local files.

Data control

The assistant is designed to be transparent and controllable, you can always view and edit data through the control panel.

Install

Install from the Reachy Mini app store

Search for CookAIware in the dashboard, install, and start talking.

  1. Make sure you have the dashboard installed.
  2. Connect to your Reachy Mini using the dashboard.
  3. Navigate to the "Applications" tab in the dashboard.
  4. Find "cookAIware" in the list of available applications.
  5. Click "Install".

Set up your OpenAI API key

Add your OPENAI_API_KEY to .env locally.

Control panel

Access local settings from the Reachy dashboard by clicking the gear icon next to the app.

From there, you can view the live conversation and optionally manage inventory, meal planning and the shopping list.
